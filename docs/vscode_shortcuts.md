# VS Code shortcuts

## Open cpmmand palette

`⌘` `⇧` `P`

## View markdown preview

`⌘` `⇧` `V`

## copy line

`⌥` `⇧` `↓` OR `⌥` `⇧` `↑`

## Move line 

`⌥` `↓` OR `⌥` `↑`


## Reveal file in explorer

`⌘` `⇧` `P` -> type `reveal` ->  choose `reveal active file in explorer view`

## Delete file

`⌘` `⌫`
